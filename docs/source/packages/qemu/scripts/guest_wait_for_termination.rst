qemu.guest_wait_for_termination
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

.. automodule:: cijoe.qemu.scripts.guest_wait_for_termination
   :members:

CLI arguments
-------------

* ``--guest_name GUEST_NAME``

  Name of the qemu guest. (default: None)

* ``--timeout TIMEOUT``

  Amount of seconds to wait for the qemu guest to terminate. (default: 60)
